### Isaac Gym

| 项目 | 状态 | 路径/说明 |
|------|------|-----------|
| **Isaac Gym Python 包** | 1.0rc4 | `/home/zhz/pan1/isaacgym/python`，editable 安装 |
| **安装目录** | 已安装 | `/home/zhz/pan1/isaacgym` |
| **本地安装包** | 192M | `/home/zhz/pan1/IsaacGym_Preview_4_Package.tar.gz` |

> Isaac Gym 已安装到 `leggedgym` 环境。因为 Isaac Gym 的二进制绑定需要找到 conda 环境里的 `libpython3.8.so.1.0`，已添加 conda 激活脚本自动设置 `LD_LIBRARY_PATH`。

安装命令记录：
```bash
cd /home/zhz/pan1
tar -xf IsaacGym_Preview_4_Package.tar.gz -C /home/zhz/pan1
cd /home/zhz/pan1/isaacgym/python
conda activate /home/zhz/pan1/leggedgym
python -m pip install -e .
python -m pip install numpy==1.23.5

# 验证
python -c "import isaacgym; from isaacgym import gymapi, gymtorch; print(isaacgym.__file__)"
cd /home/zhz/pan1/legged_gym
python legged_gym/scripts/train.py --task=anymal_c_flat --headless --num_envs=16 --max_iterations=1
```
